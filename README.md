## Features

- Classic snake gameplay with responsive controls
- Score tracking system
- Time-based movement for consistent gameplay
- Embedded sound effects
- Food generation with collision avoidance
- Easy restart functionality
- Clean, minimalist visual design

### Controls

- **↓ Arrow Down** - Move snake down
- **← Arrow Left** - Move snake left
- **Space** - Restart game after game over

## Technical Implementation

The game implements several key programming concepts:

- **Game Loop Architecture** - Standard game loop with input processing, update, and render phases
- **Time-based Movement** - Snake moves at fixed intervals (0.15s) for consistent gameplay
- **Collision Detection** - Both wall collision and self-collision detection
- **State Management** - Game state handling (playing, game over, restart)
- **Embedded Resources** - Audio files embedded directly into the executable
